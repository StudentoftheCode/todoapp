# Secure Task Manager

A full-stack task management application that allows users to securely create, manage, and organize their tasks through an authenticated account system and persistent database storage.

## Features

- User authentication and protected routes
- Secure login and registration flow
- Create, view, edit, and delete tasks
- Persistent task storage with a database
- Clean and responsive user interface
- Organized task management experience
- Session-based or token-based authentication support
- Backend API for task and user operations

## Project Purpose

This project was built to demonstrate full-stack development skills, including authentication, database integration, CRUD functionality, and secure user access control. It simulates a real-world productivity application where each user can manage their own tasks in a secure environment.

## How It Works

Users can register for an account, log in securely, and access their personal task dashboard. Once authenticated, they can create new tasks, update existing ones, and delete completed or unwanted tasks. All task data is stored in a connected database, ensuring persistence across sessions.
